# 📸 EcoSnap Cameroon v2.0
**Observatoire Numérique et Comparateur de Prix des Marchés de Yaoundé**

EcoSnap Cameroon est une application de Civic Tech conçue pour monitorer les prix des produits de première nécessité. Elle permet une collecte de terrain rapide et offre des outils d'analyse comparative multi-marchés pour aider les ménages à optimiser leur budget.

## 🌟 Fonctionnalités Clés
- **Saisie Mobile-First** : Formulaire optimisé pour les relevés en direct sur les marchés (Mokolo, Mfoundi, etc.).
- **Analyse Comparative Dynamique** : Identification automatique du marché le moins cher par produit.
- **Visualisations Avancées** : Répartition géographique (Pie Chart) et Analyse de prix (Histogrammes groupés).
- **Maintenance Intégrée** : Outils de nettoyage et de gestion des erreurs pour garantir l'intégrité de la base CSV.

## 🛠️ Installation
1. Clonez le dépôt : `git clone https://github.com/ton-pseudo/EcoSnap-Cameroon.git`
2. Installez les bibliothèques : `pip install streamlit pandas plotly`
3. Lancez l'application : `streamlit run app.py`
